    Comparative Analysis of Deep Learning Architectures for Classifying Cable Drums

    March 2024 - June 2024

    This project was conducted for Drumster, a cable drum recycling company based in Jönköping, Sweden. Drumster receives drums from customers for recycling but aims to avoid accepting non-reusable drums, as this would result in a loss for the company. The project involves a comparative analysis of two deep learning architectures, VGG16 and ResNet50, for classifying cable drums. The study focuses on improving detection accuracy by employing data augmentation techniques to address class imbalance and enhance model generalization.

    Programming Languages Used

    Python

    Frameworks and Libraries

    TensorFlow, Numpy,OS

    Evaluation Metrics

    Recall, F1 Score

    Key Works

    • Validation of image file extensions to ensure only valid images are included.
    • Conversion of images to a consistent format (RGB) and NumPy arrays.
    • Data augmentation techniques such as geometric transformations and color space augmentations to address class imbalance.
    • Training and evaluation of VGG16 and ResNet50 architectures on the augmented dataset.
    • Grad-CAM Visualization: Implemented Grad-CAM (Gradient-weighted Class Activation Mapping) to visualize and interpret the regions of the images that the models focus on when making predictions.

    Outcome

    The study found that data augmentation techniques significantly enhance the ability of deep learning models to generalize from limited training data, leading to improved performance in real-world applications. The project demonstrated the importance of robust image-based damage detection methods for cable drums, contributing to maintaining high operational standards and sustainability practices. Additionally Grad-CAM visualization provided valuable insights into the decision-making process of the models, allowing for better understanding and trust in the model's predictions.

    Evaluating Performance of Different Models in Churn Prediction: A Comprehensive Study

    January 2024 - March 2024

    The project focused on evaluating and comparing the performance of four machine learning algorithms for churn prediction using a dataset from Skogsro Spa, a company based in Jönköping, Sweden. The aim was to identify the most effective model for predicting customer churn to support e-commerce businesses in developing better customer retention strategies.

    Programming Languages Used

    Python

    ML Libraries Used

    Logistic Regression, Random Forest, Support Vector Machine (SVM), Gradient Boosting

    Evaluation Metrics

    Accuracy, Recall, Precision, F1 Score, Area Under the Curve (AUC)

    Key Works

    • Data Preparation: Cleaning and merging datasets, handling missing values, and transforming data types for accurate analysis.
    • Feature Engineering: Creation of features like purchase analysis, product purchase quantity, order frequency, promotional features, product categorization, product category preferences, recency of purchases, geographic segmentation, and churn history integration.
    • Model Development: Building and tuning four machine learning models (Logistic Regression, Random Forest, SVM, Gradient Boosting).
    • Model Evaluation: Assessing models using various performance metrics to determine the best-performing algorithm.

    Outcome

    The project revealed that Random Forest and Gradient Boosting models provided the highest accuracy in churn prediction. These findings help e-commerce businesses in identifying customers at risk of churning and optimizing retention strategies. The study supports improved resource allocation and enhances customer lifetime value by providing actionable insights into customer behavior and churn patterns.
